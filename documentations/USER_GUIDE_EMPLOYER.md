## ğŸ” Finding the Right Talent

### Search by Skills

**The Power of SkillSense**: Find candidates by actual skills, not just keywords!

**How to search**:

1. Click **"Search Talent"** on your dashboard
2. Enter skills you're looking for:
   ```
   Example: "React, TypeScript, AWS"
   ```
3. Set filters:
   - **Confidence Level**: Minimum skill confidence (e.g., 80%+)
   - **Experience Level**: Junior, Mid, Senior
   - **Location**: City, state, or remote
   - **Availability**: Immediate, 2 weeks, 1 month

**Search Results Show**:
- Candidate name (if profile is public)
- Top 5 skills with confidence scores
- Skill match percentage
- Evidence snippets

---

### Understanding Skill Confidence

**What the confidence score means**:

- **90-100%**: Expert level
  - Strong evidence from multiple sources
  - Recent, active usage
  - **Hire with confidence!**

- **70-89%**: Proficient
  - Good evidence from CV or GitHub
  - Regular usage
  - **Great candidates!**

- **50-69%**: Intermediate
  - Some evidence
  - Learning or occasional use
  - **Good for junior roles**

- **Below 50%**: Beginner
  - Limited evidence
  - May be learning
  - **Consider for entry-level**

**âœ¨ Pro Tip**: Look for 80%+ confidence for critical skills!
